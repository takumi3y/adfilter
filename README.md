# adfilter
### Twitter @adfiltertxt
広告フィルタ配布
# 注意
このフィルタは完全に広告を消せません。
セキュリティ重視です。
広告を消したいのなら[280さんのフィルタ](https://280blocker.net/files/280blocker_domain.txt)を使用して下さい。
# 再配布等について
再配布はTwitterID(@adfiltertxt)を明記して再配布して下さい。改変可能です。
商用利用は可能です。再配布のようにして下さい。
